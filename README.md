# 🥗 NutriGlow – Your Smart Nutrition Guide

NutriGlow is a web-based AI-powered chatbot that offers personalized wellness support by providing yoga routines and nutrition plans based on user goals. Whether you're aiming for weight loss, weight gain, improved nutrition, or a custom diet, NutriGlow helps you make smarter lifestyle choices.

## 💡 Features

- 🤖 **Interactive Chatbot** powered by IBM Watson Assistant
- 🧘‍♀️ Yoga recommendations for:
  - Weight Loss
  - Weight Gain
  - General Wellness
- 🥗 Personalized Diet Plans based on user goals
- 📱 Responsive Website UI using **HTML** and **Tailwind CSS**
- 📚 Easy-to-navigate flow: Choose Goal → Get Info → View Syllabus or Notes

## 🛠 Tech Stack

- **Frontend**: HTML, Tailwind CSS
- **Chatbot**: IBM Watsonx Assistant
- **Hosting**: IBM cloud

## 🚀 How It Works

1. **User visits the website** and is greeted by the NutriGlow chatbot.
2. The chatbot offers four primary goals:
   - Yoga for Weight Loss
   - Yoga for Weight Gain
   - Yoga for Better Nutrition
   - Personalized Healthy Diet Plan
3. The user selects a goal, and the chatbot responds with:
   - Recommended yoga asanas
   - A brief explanation or benefits
   - A video link to practice that exercise
4. The chatbot guides users with friendly interactions and relevant tips.
